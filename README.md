# AutoTest-MSSQL
Preconditions:
[Database Mail for SQL Server](https://www.mssqltips.com/sqlservertip/1100/setting-up-database-mail-for-sql-server/)

Configuration for Gmail: 
* SMTP server name: smtp.gmail.com
* Port: 587
* SSH: Enabled
